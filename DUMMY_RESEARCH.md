Dummy Decomposition Prompt
==========================

A cheap prompt to test if decomposition & publishing etc works without wasting 30min and $10 :D

Make sure to pick `Expedition`!

```txt
Smoke-test for the decompose pipeline. Three independent micro-decisions, each
decision-only with a single citation. Please decompose into three (recon) sub-topics and
do not expand any of them.

Current stable Bun version and its 2026 release date.
Current stable Caddy version and which ACME challenge type it defaults to in 2026.
Current stable ripgrep version available via Homebrew in 2026.
These are unrelated fact lookups — don't merge them into one survey.
```
