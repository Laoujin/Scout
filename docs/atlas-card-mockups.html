Atlas — card mockups

Seven card variants, four index layouts. Each card consumes the same frontmatter (title, summary, tags, depth, format, citations, reading_time_min, date) plus a 3:2 cover.* image or typographic fallback. Open-ended — tell me which parts to keep and which to cut.

V1 · Image-top card

The itenium-style default. 3:2 cover, title, one-line summary, tag chips, meta row. Depth badge top-right corner of the body. Read as: "what is this, at a glance, and is it worth opening?"

Cover art pipeline. These placeholders are CSS gradients + a 3-letter mark derived from the slug.
